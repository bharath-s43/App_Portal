After the un-assignment, new owner of the beacons will be the admin who has performed the un-assignment.

For e.g,

If Applause admin performs this task -- beacons will no longer be assigned to employee, location, brand or customer, they will be owned by Applause.

If Location admin performs this task -- beacons will no longer be assigned to employee, they will be owned by Location admin.

 

If a user does not own any of the beacons in the list, un-assignment will return an error.

For e.g

Location admin of LOCA trying to un-assign beacons from LOCB will result in error.
